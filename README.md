#  IPL Auction Data Collection, Cleaning & Exploratory Data Analysis

##  Project Overview

This project focuses on collecting, cleaning, transforming, and analyzing IPL Auction and Player Performance data from 2013–2026. The objective is to build a reliable analytics-ready dataset that can be used to evaluate player performance, auction trends, and franchise investment strategies.

The project follows a complete Data Analytics workflow including data collection, preprocessing, feature engineering, exploratory data analysis (EDA), and insight generation.

---

##  Business Objective

IPL franchises invest millions of rupees during player auctions every season. This project aims to prepare high-quality data that can later be used to answer important business questions such as:

* Which teams spend the most during auctions?
* How have auction prices evolved over time?
* Which player roles attract the highest investments?
* What are the trends in player performance?
* How can auction spending be connected to on-field performance?

---

##  Datasets Used

### IPL Auction Dataset (2013–2026)

Contains:

* Player Name
* Sold Price
* Team Name
* Role
* Nationality
* Base Price
* Capped/Uncapped Status
* Auction Year

### IPL Match & Ball-by-Ball Dataset

Used to generate player performance metrics such as:

* Runs Scored
* Balls Faced
* Strike Rate
* Fours & Sixes
* Wickets Taken
* Overs Bowled
* Economy Rate
* Matches Played

---

##  Project Workflow

### 1️ Data Collection

* Gathered IPL auction datasets from multiple years
* Combined yearly auction files into a unified dataset
* Collected IPL match and ball-by-ball performance data

### 2️ Data Cleaning

* Removed unnecessary columns
* Handled missing values
* Standardized team names
* Corrected inconsistent player information
* Converted price columns into numeric format

### 3️ Feature Engineering

Created new analytical features including:

* Performance Score
* Strike Rate
* Economy Rate
* Matches Played
* Boundary Statistics
* Batting & Bowling Metrics

### 4️ Exploratory Data Analysis (EDA)

Performed detailed analysis on:

* Auction Spending Trends
* Team-wise Investments
* Role-wise Spending
* Nationality Analysis
* Top Run Scorers
* Top Wicket Takers
* Best Strike Rates
* Best Economy Rates
* All-Rounder Analysis

---

##  Key Insights

* Auction spending has increased significantly over the years.
* All-Rounders attract the highest investments across franchises.
* Certain franchises consistently invest more aggressively in auctions.
* Performance metrics reveal significant differences between player roles.
* Several high-performing players were acquired at relatively lower prices.
